## Sending the Message
The following code will prompt the message with the pre-filled fields to the user to send it:

<div class="code-switcher show-cpp-true">
<div class="switcher" >
<span class="sw-bp" onclick="switchBp()">Blueprints</span><span class="sw-cpp" onclick="switchCpp()">C++</span>
</div>
<div class="cpp">

```cpp
#include "UltimateIOSToolkit/UI.h"

void UMyClass::SendMessage()
{
	UIT::UI::SendMessage
	(
		// The numbers to send the message to.
		{ TEXT("0123456789"), TEXT("0987654321") },
		
		// The subject of the message. Not always supported and might be ignored. 
		// Call UIT::UI::CanSendSubject() to check availability.
		TEXT("The subject"),
		
		// The message's content.
		TEXT("This is the body of the message."),
		
		// If we want to disable user attachments.
		true,
		
		// An optional callback called when the overlay is dismissed.
		FOnMessageSent::CreateLambda([](EIOSMessageComposeResult Result) -> void
		{
			switch(Result)
			{
			case EIOSMessageComposeResult::Sent:      /* Message sent.          */ break;
			case EIOSMessageComposeResult::Cancelled: /* Cancelled by the user. */ break;
			case EIOSMessageComposeResult::Failed:    /* An error occurred.     */ break;
			}
		})
	);
}
```
