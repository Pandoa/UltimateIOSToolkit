## Example 
For this example, let's imagine we want to prevent a player to start a match if his battery level is lower than 10% and is not currently charging.  
The following code prevents the player to join the match in such cases.

```cpp
#include "UltimateIOSToolkit/Device.h"

void UMyClass::StartMyMatch()
{
    // First, check if we have access to the battery.
    if (UIT::Device::IsBatteryMonitoringEnabled())
    {
        // A boolean indicating if the device passes the battery requirements.
        const bool bHasEnoughBatteryForMatch = 
            UIT::Device::GetBatteryLevel() > 0.1f ||                           // More than 10% of battery life.
            UIT::Device::GetBatteryState() == EIOSDeviceBatteryState::Plugged; // Or currently charging

        // If the requirements aren't met.
        if (!bHasEnoughBatteryForMatch)
        {
            // Prevent the user from starting/joining the match and ask to load the device.
            return;
        }
    }

    else 
    {
        // No battery monitoring. Make sure to call UIT::Device::SetBatteryMonitoringEnabled(true).
    }

    // The user has more than 10% battery, let's start the match.
}
```
