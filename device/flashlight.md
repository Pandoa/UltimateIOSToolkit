## Turn On the Flashlight

You can turn the flashlight on with the following code.

```cpp
#include "UltimateIOSToolkit/Device.h"

void UMyClass::TurnOnFlashlight()
{
    // Toggle the flashlight.
	UIT::Device::SetFlashlightEnabled(true);

    // Or set directly a custom brightness.
	UIT::Device::SetFlashlightBrightness(0.7f);
}
```
